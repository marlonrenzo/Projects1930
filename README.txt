Top-level files:
index.html is the landing page
login.html allows the user to log in/ sign up
map.html holds all the embedded google map locations of retailers we've derived out database from
productListing.html is used to display our database of items depending on the category selected
shoppingcart.html helps the user keep track of items they're interested in
WIP.html is a temporary placeholder page for features still yet to be implemented
SansationRegular.woff is an external font file
firebase.json and firebaserc is for firebase usage

Next-level files:
css folder holds external stylesheets for each page
images folder holds external images integrated on our landing page 
js folder holds our extracted javascript files
.firebase folder is used for hosting cache
.git folder is for github compatability
